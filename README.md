## Getting Started

### Prerequisites

- [Git](https://git-scm.com/)
- [Node.js and npm](nodejs.org) Node ^4.2.3, npm ^2.14.7

### Developing

1. Run `npm install` to install server dependencies.

2. Run `node index` to start the development server.
